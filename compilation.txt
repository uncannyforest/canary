These instructions assume you have the java SDK installed on your system


TO COMPILE TERM PROJECT
-----------------------

In a console, navigate to the termproject directory, and then type

javac gui\Canary.java

Then, to open the program, run

java gui\Canary



TO COMPILE A HOMEWORK
---------------------

In a console, navidate to the homework directory, and then type
javac <path to homework main file>

Then to open the program, run
java <path to homework main class>

In most folders, there is only one file, the main file.  Here are the homeworks that are different:
HW00 is in the top level of the homework directory.
HW01 is in a folder by itself called "DrunkenWalk"
HW09 has two java files.  The main class is Mandelbrot.java
